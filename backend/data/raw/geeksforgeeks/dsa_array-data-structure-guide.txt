Note that in case of C/C++ and Java-Primitive-Arrays, actual elements are stored at contiguous locations. And in case of Python, JS, Java-Non-Primitive, references are stored at contiguous locations. It offers mainly the following advantages over other data structures.Random Access :  i-th item can be accessed in O(1) Time as we have the base address and every item or reference is of same size.Cache Friendliness : Since items / references are stored at contiguous locations, we get the advantage of locality of reference.Arrays are used to build other data structures like Stack Queue, Deque, Graph, Hash Table, etc.  An array is not useful in places where we have operations like insert in the middle, delete from middle and search in a unsorted data.
